get_Post
--------

### Synopsis
Gets the posts in a site.

---

### Description

Gets the post files in a Jekyll site.

Any file in the _posts directory is considered a post.

---

### Related Links
* [https://jekyllrb.com/docs/posts/](https://jekyllrb.com/docs/posts/)

---

### Examples
> EXAMPLE 1

```PowerShell
$psJekyll.CurrentSite.Post
```
